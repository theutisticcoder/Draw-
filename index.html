The Physics of Wet Pickup

Most paint simulation websites draw static static-colored pixels over a canvas. L'Atelier's new engine is built around a dynamic brush head memory system:

Real-time Pigment Absorption

As the brush drags across existing paints, it continuously absorbs underlying hues into its buffer, polluting the active stroke dynamically.

Viscous RGB Fluid Mix

Viscous brush attributes average structural values cleanly across the drawing array to allow natural color transitions.

RGB Fluid Specifications viscous blend

Viscous blending calculations perform weighted averages directly over sample matrices to capture high fidelity brush dynamics:

// Viscous brush dynamic mapping
weight = brushWetness * 0.45;
blended.r = tip.r * (1 - weight) + canvas.r * weight;
blended.g = tip.g * (1 - weight) + canvas.g * weight;
blended.b = tip.b * (1 - weight) + canvas.b * weight;

This maintains excellent responsiveness, avoiding black decay loops entirely.
