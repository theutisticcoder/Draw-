Subtractive vs Additive:
Why RGB is muddy.

Standard digital screens mix colors additively (Red + Green = Yellow, Blue + Yellow = Gray). Physical oil paints mix subtractively because pigment particles absorb specific wavelengths of light.

Our specialized conversion maps RGB components to a custom RYB space, interpolates the density properties, and resolves the correct reflected color spectrum back to your screen.

Dynamic Reservoir Pickup

As your brush moves through wet paint, it collects pigment off the canvas, continuously modifying the brush color until the loaded reservoir runs clean or is re-dipped.

The Physics of Wet Pickup

Most paint simulation websites draw static static-colored pixels over a canvas. L'Atelier's new engine is built around a dynamic brush head memory system:

Real-time Pigment Absorption

As the brush drags across existing paints, it continuously absorbs underlying hues into its buffer, polluting the active stroke dynamically.

Subtractive Color Math (RYB Model)

Colors are translated mathematically into Red, Yellow, and Blue vectors. They are blended subtractively and re-converted, yielding pristine Greens, Oranges, and Violet tones.

RYB Converter Specifications Subtractive active

Conversion from standard RGB coordinates to Subtractive RYB space is executed using trilinear interpolation across a custom color cube:

// Interpolating color vectors
f_000 = rgb(1, 1, 1)        // White
f_100 = rgb(1, 0, 0)        // Pure Red
f_010 = rgb(1, 1, 0)        // Pure Yellow
f_001 = rgb(0, 0.3, 1)      // Pure Blue
f_110 = rgb(1, 0.5, 0)      // Orange (R+Y)
f_011 = rgb(0, 0.7, 0.2)    // Green (Y+B)
f_101 = rgb(0.5, 0, 0.5)    // Violet (R+B)

This ensures physical combinations match real life artist palette logic perfectly.
